Opdracht 2.1 - 1, 2, 3 Feature Detection

HTML

Srcset attribute

Wordt niet ondersteund door

  • IE 8, 9, 10 & 11
  • Opera Mini 8
  • Android Browser 4.3, 4.4 & 4.4.4

Fallback

  • Als de browser de srcset attrubute niet kan lezen pakt hij de src attribute

input type=date

Wordt niet ondersteund door

  • Firefox
  • Safari

Fallback

  • Als de browser type="date" niet ondersteund wordt de placeholder="dd-mm-jj" weergegeven

CSS

Linear-gradient

Wordt niet ondersteund door

  • IE 8 & 9
  • Opera Mini 8

Fallback

  • background-image: url(img/buttonBackground.png);
  • background-color: #3498DB;

RGBA

Wordt niet ondersteund door

  • IE 8

Fallback

  • background-color: #99CCED;
