Github

...

Dinsdag 17 mei 2016


Sinds kort zit Github ook gevestigd in Amsterdam en wij mochten daar even komen kijken. Er werd een zeer interessant verhaal verteld over de werking van Github en hoe waardevol het is voor de samenwerking met collega’s in het buitenland. Het is natuurlijk ook best lastig samenwerken als je collega slaapt als jij aan het werk bent. Als je samen aan één project werkt voegt Github alle files samen en zorgt ervoor dat alle aanpassingen bij iedereen doorgevoerd worden. Alleen soms snapt Github het even niet meer en dan ontstaan er Merging Conflics. Dit is erg vervelend want dat kan je de files weer aflopen want Github heeft dan op sommige plaatsen stukken dubbel geplaats en dan moet je handmatig datgeen weghalen wat niet meer van toepassing is. Deze conflics kunnen nog aardig wat tijd in beslag nemen.

Naast Github gebruiken ze ook Slack om te communiceren. Zij vinden dat erg handig omdat iedereen kan zien wie wat aan het doen is. En als er echt een specifiek bericht naar jou gericht is kan je gebruiken maken van het hashtag (#) teken met daaropvolgend de naam van de persoon aan wie het bestemd is. Het voordeel daarvan is, is dat iedereen dan kan meelezen en daar dan ook weer van kan leren.

Geschreven door: Linda van Dijk

Datum: 30 mei 2016
