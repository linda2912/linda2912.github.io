CSS animaties & transities

In dit artikel ga ik laten zien hoe makkelijk animeren is met CSS waar voorheen een programma zoals Flash of een andere taal als Javascript voor nodig was. Tegenwoordig kan dit gemakkelijk met alleen HTML en CSS, dit scheelt veel regels code.

Eerst zal ik wat meer vertellen over de twee begrippen animaties en transities

CSS Transities

Met CSS transities kun je de waardes van elementen veranderen in de loop van een bepaalde tijd. Er zijn vier verschillende eigenschappen.

transition-property

Met transition-property kun je aangeven voor welk element de transitie bedoeld is.

transition: width;

transition-duration

Specificeert hoe lang de transitie moet duren.

transition: width 1s;
