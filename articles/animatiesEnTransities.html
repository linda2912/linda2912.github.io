CSS animaties & transities

Geschreven door Linda van Dijk Op 24 juni 2016

In dit artikel ga ik laten zien hoe makkelijk animeren is met CSS waar voorheen een programma zoals Flash of een andere taal als Javascript voor nodig was. Tegenwoordig kan dit gemakkelijk met alleen HTML en CSS, dit scheelt veel regels code. Eerst zal ik wat meer vertellen over de twee begrippen animaties en transities

CSS Transities

Met CSS transities kun je de waardes van elementen veranderen in de loop van een bepaalde tijd. Er zijn vier verschillende eigenschappen.

transition-property

Met transition-property kun je aangeven voor welk element de transitie bedoeld is.

transition: width | opacity | all;

transition-duration

Specificeert hoe lang de transitie moet duren.

transition: 1s | 14ms;

See the Pen transition-duration by Linda (@Lindavandijk) on CodePen.

transition-timing-function

Specificeert de snelheidscurve van de transitie. Met cubic-bezier kun je zelf een curve creeën. Ook kun je kiezen uit standaard curven op Easing Functions Cheat Sheet

transition-timing-function: ease | ease-in | ease-out | ease-in-out | linear | step-start | step-end | cubic-bezier(0.1, 0.7, 1.0, 0.1) | steps(4, end);

See the Pen transition-timing-function by Linda (@Lindavandijk) on CodePen.

transition-delay

Specificeert na hoeveel seconde de transitie moet beginnen.

transition-delay: 1s | 14ms;

See the Pen transition-delay by Linda (@Lindavandijk) on CodePen.

CSS animaties

Met CSS animaties kun je verschillende transities achter elkaar uitvoeren. Dit maakt het mogelijk om hele vette animaties te maken met relatief weinig code. CSS animaties maak je met keyframes en animations

animation

Animation eigenschappen worden gebruikt om @keyframes toe te wijzen aan een specifiek element om te bepalen hoe het element wordt geanimeerd.

body { animation: element; }

keyframes

Keyframes worden gebruikt om de stijlen aan een element op verschillende tijden te definiëren.

@keyframes element { .. }

De sub-eigenschappen van de animatie-eigenschap zijn:

animation-delay

Geeft aan na hoeveel seconde de animatie moet beginnen.

animation-delay: 1s | 14ms;

animation-direction

Geeft aan in welke richting een animatie om gaan.

animation-direction: normal | reverse | alternate | alternate-reverse;

animation-duration

Geeft aan hoe lang een animatie moet duren.

animation-duration: 6s | 120ms | 1s, 15s | 10s, 30s, 230ms;

animation-fill-mode

Specificeerd een stijl voor het element als de animatie klaar is of nog niet begonnen is.

animation-fill-mode: none | forwards | backwards | both;

animation-iteration-count

Geeft aan hoeveel keer de animatie moet afspelen. Dit kan een nummer zijn of eindeloos (infinite).

animation-iteration-count: 1 | 23 | infinite | 2, 0, infinite;

animation-name

Specificeerd de naam van de @keyframes animatie.

animation-name: eigenNaam;

animation-play-state

Bepaalt of de animatie wordt uitgevoerd of is onderbroken.

animation-play-state: running | paused;

animation-timing-function

Specificeert de snelheidscurve van de animatie.

animation-timing-function: ease | ease-in | ease-out | ease-in-out | linear | step-start | step-end | cubic-bezier(0.1, 0.7, 1.0, 0.1) | steps(4, end);

Voorbeeld

Om een animatie te maken moet je eerst bepalen welk element je wilt animeren. Vervolgens zet je op dat element de CSS stijl animation: animeerDitElement;. Deze naam mag je zelf verzinnen en ga je straks gebruiken om hem met @keyframes aan te roepen. Het is makkelijker en sneller om andere eigenschappen in dezelfde regel te definiëren en komt er dan zo uit te zien:

animation: animeerDitElement 1s ease;
Vervolgens ga je met @keyframes de animatie maken. Dit zal er ongeveer zo uit zien: 
@keyframes animeerDitElement {
	0% { .. }
	100% { .. }
}

Nu kun je bepalen welke transities de animatie achter elkaar moet doen.

See the Pen animation-delay by Linda (@Lindavandijk) on CodePen.

Zoals je ziet kun je op een vrij makkelijke manier animaties maken. Als je dit toepast op je website, wordt je website net even mooier en aantrekkelijker om naar te kijken. Vaak maakt een animatie het ook sneller duidelijker wat de gebruiker ergens mee kan. Als bijvoorbeeld een hamburger menu veranderd in een kruisje, is het direct duidelijk dat daar het menu weer gesloten kan worden. Als dit gedaan wordt met een animatie valt het direct op voor de gebruiker en ziet hij sneller dat het icoontje veranderd is.

Conslusie

Het implementeren van CSS transities en animaties vereist geen hoge technische kennis maar als je hele gecompliceerde animaties wilt maken is er wel enige ervaring vereist. Het heeft een grote toegevoegde waarde voor je website om animaties te gebruiken voor kleine elementen zodat je het voor de gebruiker makkelijker maakt om door jouw website heen te gaan.
