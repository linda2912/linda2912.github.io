CSS animaties & transities

In dit artikel ga ik laten zien hoe makkelijk animeren is met CSS waar voorheen een programma zoals Flash of een andere taal als Javascript voor nodig was. Tegenwoordig kan dit gemakkelijk met alleen HTML en CSS, dit scheelt veel regels code.

Eerst zal ik wat meer vertellen over de twee begrippen animaties en transities

CSS Transities

Met CSS transities kun je de waardes van elementen veranderen in de loop van een bepaalde tijd. Er zijn vier verschillende eigenschappen

transition-property

Specificeert de naam waar de transitie voor bedoeld is.

transition-duration

Specificeert hoe lang de transitie duurt.

See the Pen transition-duration by Linda (@Lindavandijk) on CodePen.

transition-timing-functionp

Specificeert de snelheidscurve van de transitie.

See the Pen transition-timing-function by Linda (@Lindavandijk) on CodePen.

transition-delay

Specificeert na hoeveel seconde de transitie moet beginnen.

See the Pen transition-delay by Linda (@Lindavandijk) on CodePen.
